## 项目介绍
本项目是一个django-rest-framework的模板项目，可以直接克隆后基于本项目做二次开发。

项目使用 python3.10、django4.0、django-rest-framework3.13 进行开发。

项目提供的功能：
- 短信验证码注册登录
- app: 人员信息收集示例
- app: 文章发布示例
- 支付宝接口


## 项目安装
```shell
mkdir .venv
pipenv install
```

## 依赖安装
```shell
# 在Linux系统中，oss2包需要python-devel依赖，不然会导致上传下载效率非常低
apt-get install python-dev

```
